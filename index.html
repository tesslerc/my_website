03 About

A short note
on the work.

I’m interested in the space between a model that can move and a machine that can make the next decision.

My work focuses on learning reusable motion priors and using them to solve near-short horizon control problems. The goal is a system that carries useful behavior forward — not a policy that only works once, in one place.

This is a working archive. The publication list below keeps the research trail together; code and experiments live on GitHub ↗.
